VIM TRIX (Julie)

Install Pathogen to install vim plugins easily.

good plugins

  • Sleuth: detects tabbing in file, and sets your settings to that
  • Syntastic: Syntax checking in vim
  • nerd commenter: comment and uncomment code
  • luna vim: pretty color scheme, installable thru pathogen :)
  • sensible: nice vim defaults to have

" indentation guides
" for tabs
set list listchars=tab:‧\ ,trail:·,extends:»,precedes:«,nbsp:×

let g:indentLine_char = '│'

set tw=80 " limit the length of lines to ~80 characters
Remember: visual select, then
  • = will indent your code
  • gq will split your codes into lines as long as the tw property (text wrap)
